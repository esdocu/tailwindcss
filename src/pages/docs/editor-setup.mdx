## Soporte de sintaxis

Tailwind CSS usa muchas reglas personalizadas de CSS como `@tailwind`, `@apply` y `@screen`, y en muchos editores esto puede desencadenar advertencias o errores cuando estas reglas no se reconocen.

La solución a esto es casi siempre instalar un complemento para tu editor/IDE para soporte de lenguaje PostCSS en lugar de CSS normal. Por ejemplo, VS Code tiene un complemento [PostCSS Language Support](https://marketplace.visualstudio.com/items?itemName=csstools.postcss) que puedes usar y que funciona muy bien con Tailwind CSS.

En algunos casos, es posible que debas deshabilitar las validaciones/linting nativas de CSS si tu editor es muy estricto con la sintaxis que espera en tus archivos CSS.


## IntelliSense para VS Code

La extensión oficial [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) para Visual Studio Code mejora la experiencia de desarrollo de Tailwind al proporcionar a los usuarios funciones avanzadas como autocompletar, resaltado de sintaxis y linting.

<img src={require('@/img/intellisense.png').default} alt="Tailwind CSS IntelliSense extension for Visual Studio Code" width={1344} height={672} className="rounded-lg" />

- **Autocompletar**. Sugerencias inteligentes para nombres de clases, así como [funciones y directivas CSS](/docs/functions-and-directives).
- **Linting**. Resalta errores y bugs potenciales tanto en tu CSS como en tu marcado.
- **Vistas previas al pasar el mouse**. Ve el CSS completo para un nombre de clase de Tailwind pasando el cursor sobre él.
- **Resaltado de sintaxis**. Proporciona definiciones de sintaxis para que las características de Tailwind se resalten correctamente.

Consulta el proyecto [en GitHub](https://github.com/tailwindcss/intellisense) para obtener más información, o [agrégualo a Visual Studio Code](vscode:extension/bradlc.vscode-tailwindcss) para comenzar ahora.

## JetBrains IDEs

Los IDE de JetBrains como WebStorm, PhpStorm y otros incluyen compatibilidad con las finalizaciones inteligentes de Tailwind CSS en tu HTML y cuando se usa `@apply`.

[Obtén más información sobre la compatibilidad con Tailwind CSS en los IDE de JetBrains &rarr;](https://www.jetbrains.com/help/webstorm/tailwind-css.html)
